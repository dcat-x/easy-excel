## [1.0.0] - 2026-01-12

### Added

- Initial release with OpenSpout v4 support
- Export to xlsx, csv, ods formats
- Import from xlsx, csv, ods formats
- Chunk processing for large files
- Flysystem integration for storage
- Custom headings support
- Multiple sheets support

### Changed

- Migrated from box/spout to openspout/openspout v4
- Upgraded to PHP 8.2+ requirement
- Upgraded to league/flysystem v3
